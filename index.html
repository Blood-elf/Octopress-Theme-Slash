Slash, a minimal theme for Octopress

Overview

A one-column theme for Octopress. Support all features in classic theme and include Fancybox. Do you want a different theme? That's is!

Layouts

Slash supports responsive layouts so that users can get a great visual experience. You can try this by resizing your browser at once.
